Git中，每次commit提交都会生成一个历史纪录。使用 git log 查看commit历史：

每个记录都有一个与之对应的commit id，所以可以使用命令git reset --hard commit_id来回退到相应的版本。除此之卡，在Git中，使用HEAD来代表当前版本，如需回退到前一个版本，可以使用命令git reset --hard HEAD^，前两个版本则用HEAD~2表示，以此类推。
