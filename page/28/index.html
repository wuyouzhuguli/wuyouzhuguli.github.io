Git状态跟踪

2017-06-12

对于任何一个文件，在 Git 内都只有三种状态：已提交（committed），已修改（modified）和已暂存（staged）。

1.已提交：表示该文件已经被安全地保存在版本库中了。

2.已修改：表示修改了某个文件，但还没有提交到暂存区。

3.已暂存：表示把已修改的文件已经放到暂存区了，下次提交时一并被保存到版本库中。
